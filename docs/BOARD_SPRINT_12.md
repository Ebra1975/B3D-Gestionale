# Board Sprint 12 - Gestione Template DOCX Da Interfaccia

## Obiettivo

Rendere i template DOCX gestibili dalla sezione **Documenti**, senza usare l'admin tecnico o modificare file a mano.

## Problema Pratico Risolto

L'operatore deve poter sostituire o aggiornare il modello Word usato per preventivi cliente e schede interne quando cambia layout, versione o impostazione grafica.

## Incluso Nello Sprint

- Caricamento template DOCX da interfaccia.
- Modifica dati template esistenti.
- Attivazione e disattivazione template.
- Un solo template attivo per tipo documento.
- Download del file template caricato.
- Priorita ai template personalizzati attivi rispetto ai template base generati dal gestionale.
- Test automatici su upload, validazione formato, attivazione e selezione template.

## Non Incluso

- Editor visuale dei template Word.
- Validazione completa dei segnaposto dentro il DOCX.
- Anteprima PDF direttamente dalla pagina template.
- Gestione avanzata versioni con storico approvativo.
